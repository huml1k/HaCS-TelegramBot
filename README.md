# HaCS-TelegramBot

# Настройка и запуск бота:

### В файле appsettings нужно подставить свои значения:

  1. В поле BotToken ставим свой ключ на бота.
  2. В поле BotWebhookUrl ставим свой Url.

###  Как получить свой Url:

Для начала качаем ngrok(https://dashboard.ngrok.com/get-started/setup/windows), чтобы его установить нужен впн без него никак(аналоги не работают из за политики телеграмма).
По инстуркции, которая будет на сайте активируем его и все теперь пользуемся им. 
Теперь, чтобы он прослушивал наше приложение необходимо прописать команду в Powershell или CMD: ngrok http --host-header=localhost https://localhost:7058.
После этого нам дают локальный Url, он нам и нужен. Вставляем его теперь в наше поле BotWebhookUrl.

### Запускаем проект:

В открытом Swagger и нажимаем на эндопинт setWebhook(обязательно чтобы вышло сообщение что мы установили его).

Все теперь можно пользоваться ботом.(впн должен быть всегда включен иначе ngrok прекратит работу)
